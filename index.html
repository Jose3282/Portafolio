Conocimientos

Conocimientos adquiridos

Figma

Herramienta de diseño de interfaces de usuario (UI) y experiencia de usuario (UX) basada en la nube. Permite a los diseñadores crear prototipos interactivos, colaborar en tiempo real y crear diseños y assets de alta calidad.

HTML

Lenguaje de marcado estándar utilizado para crear páginas web y aplicaciones web. Proporciona la estructura y el contenido de una página web mediante el uso de elementos y etiquetas que definen cómo se deben mostrar los diferentes elementos en un navegador.

CSS

Lenguaje de estilo utilizado para describir la presentación y el aspecto visual de un documento escrito en lenguaje de marcado como HTML o XML. CSS separa el contenido de un documento de su diseño y permite controlar cómo se visualiza en diferentes dispositivos y medios.

JavaScript

Lenguaje de programación de alto nivel, interpretado y orientado a objetos, principalmente utilizado para el desarrollo de aplicaciones web interactivas.

React

Biblioteca Javascript de código abierto diseñada para crear interfaces de usuario con el objetivo de facilitar el desarrollo de aplicaciones en una sola página.
